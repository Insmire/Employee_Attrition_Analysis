Attrition Prediction

About

The purpose of this project is to determine the possible variables that would influence an employee to leave their workplace and the relative importance of these features. We also aim to utilize a machine learning model to predict the probability that an employee will decide to leave. This is accomplished by saving and retrieving the model through serialization and deserialization, creating a Flask app to retrieve an employee's information and handle the data. Furthermore, we conducted exploratory data analysis to detect trends from each feature as they relate to the target of attrition.
